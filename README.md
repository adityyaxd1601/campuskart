# CampusKart

A student-only marketplace where college students can buy, sell, rent, and exchange products within their campus community.

## Features

- Student Verification
- Product Listings
- Search & Categories
- In-App Chat
- Create Listing
- Responsive Design

## Product Management Deliverables

- User Research
- Competitive Analysis
- User Personas
- MVP Definition
- Wireframes
- Success Metrics
- Product Roadmap
